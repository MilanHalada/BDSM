Chrome tab bar labels

Shown on each tab group in the browser. Group cards below always show order numbers.

Default keys: Alt+Shift+Up/Down cycles workspaces, Left/Right cycles tabs in the active group. Chromium only allows four predefined shortcuts per extension, so jump to workspace 1–9 and 10 use commands switch-workspace-1 … switch-workspace-9 and switch-workspace-0 (10th slot) — bind e.g. Alt+Shift+digits under chrome://extensions/shortcuts; order matches numbered titles on the tab strip.

Auto group rules

When a tab URL matches a rule’s regex, that tab joins the target group — the strip moves that workspace to the end, expands it, and collapses others. Rules are evaluated top to bottom — first match wins.
